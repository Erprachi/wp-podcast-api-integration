# WordPress API Integration: Dynamic Podcast Aggregator

## 🚀 The Project
Is project mein maine Jim Cummings ke podcasts ko dynamically fetch karne ke liye custom PHP logic aur API integration ka use kiya hai.

## 🛠 Technical Details
* **API Fetching:** External server se data load karne ke liye custom backend script.
* **Templates:** 
    * `Main Page`: Saare episodes ki listing handle karta hai.
    * `Single Page`: Har episode ka unique detail view generate karta hai.
* **Performance:** API calls ko optimize kiya gaya hai taaki site ki speed maintain rahe.

## 💻 How to use
In templates ko WordPress theme directory mein use kiya ja sakta hai dynamic content render karne ke liye.
